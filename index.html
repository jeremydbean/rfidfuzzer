RFID Fuzzer Playlist Generator

You can generate card codes by:

  1. Ranges: Enter ranges in the Card Number Ranges field.
  2. Anchors: Select base card numbers below and an Anchor Count; the script will take those base IDs and include sequential offsets.
  3. Both: Combine ranges and anchors to include codes from both sources.

Anchor Numbers: Pick specific bases and count to include sequential offsets. The first quarter of that count for each selected base will be listed first, followed by randomized offsets.
